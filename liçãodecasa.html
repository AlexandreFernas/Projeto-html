Estrutura básica do HTML

Para chegar na conclusão de uma página de HTML assim como qualquer receita alimentar temos que usar os ingredientes que somados da maneira correta atingem no objetivo principal. Para montar a base de uma página de HTML precisamos desses comandos em sequencia: "HTML, HEAD e BODY". O comando "HTML" é para página identificar que a estrutura será baseado no "HTML". O comando "HEAD" é a função que vai carregar informações antes de exibir para o usúario. Por sua vez o comando "BODY" é o corpo da página "HTML" é o que vai ser mostrado para quem acessa a página.

Tags

As tags são funções que criam ou alteram a formatação do corpo de HTML, como por exemplo a tag "i" que deixa o texto em italico. "strong" deixa o texto em negrito. Algumas tags não possuem a tag de fechamento, como por exemplo o "IMPUT" que individualmente já é um campo e a sua função é de criar uma barra onde se pode inserir tanto letras como números. Já a tag "IMG" permite anexar uma imagem ao corpo da página. Alguns comandos são específicos de algumas tags, como por exemplo "Style" onde a partir dessa ordem é possível alterar a cor da fonte.

Textos

Como comentado anteriormente, dentro da funcionalidade do HTML é possível fazer diversas modificações como por exemplo alterar a cor da fonte. Esse comando é específico dessa função sobre textos. Existem outros comandos especialmente para textos, como por exemplo "H1, H2, H3, H4" que alteram o tamanho da fonte. A tag "P" insere um parágrafo. "blockquote" serve para citar ou colocar uma observação dentro do espaço escolhido.

Listas ordenadas e não ordenadas

De modo resumidamente e simples, as listas ordenadas e não ordenadas servem para a criação de tópicos ou subitens. Diferentemente das ordenadas a listas não ordenadas servem para dizer ao navegador que não existe um ordem correta na lista em questão. A tag que corresponde as listas ordenadas e não ordenadas é respectivamente "OL" e "UL".

Links

Os links dentro do HTML são correspondidos pela tag "A" e sua função é de linkar a página html com outra página podendo ela ser de outro site ou não ou apenas linkar com uma parte do próprio corpo estruturado. Para usar outra aba para entrar em outro site é preciso usar a função "TARGET".
